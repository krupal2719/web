4.what is html entities?withexample.

Reserved characters in HTML must be replaced with character entities.
A commonly used entity in HTML is the non-breaking space:   A non-breaking space is a space that will not break into a new line.
Examples:-

  • § 10

  • 10 km/h

  • 10 PM


Another common use of the non-breaking space is to prevent browsers from truncating spaces in HTML pages. If you write 10 spaces in your text, the browser will remove 9 of them. To add real spaces to your text, you can use the   character entity.

14.what is the use of a span tag?explain with example.

The span tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute.

EXAMPLE:- The span tag is much like the div element, but div tag is a block-level element and span is an inline element.

20.what are logical and physical tags in HTML?
Physical and Logical tags are used in HTML for better visibility and understanding of the text by the user on the web page. However, both tags differ from each other as suggested by their names.
Logical Tags:-
These tags are used to provide the visual appearance to the text.
Physical Tags:-
These tags are used to add some logical or semantic value to the text.
